# Great Filmmakers

Twelve filmmaker personas (6 directors + 2 writers + 4 craft specialists) plus slash commands for scene breakdown, multi-discipline critique, and end-to-end film treatment generation. A Claude Code plugin. Third in the Caseproof persona trilogy:

- [`great-minds-plugin`](https://github.com/sethshoultes/great-minds-plugin) — strategic decision-makers
- [`great-authors-plugin`](https://github.com/sethshoultes/great-authors-plugin) — prose craft
- **`great-filmmakers-plugin`** (this repo) — film craft

**Status:** Design phase. See [`docs/superpowers/specs/2026-04-24-great-filmmakers-design.md`](docs/superpowers/specs/2026-04-24-great-filmmakers-design.md) for the full spec.
